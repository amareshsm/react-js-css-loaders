# react-js-css-loaders

React CSS Loaders

## Quick Start: ##

#### To install package: ####

using npm:

```
npm i react-js-css-loaders 
```
   
using yarn:

```
yarn add react-js-css-loaders 
```   
  
### To import Loaders: ###

```
import {Moon} from 'react-js-css-loaders'

const LoaderComponent = (props) => (
  <div>
    <Moon />
  </div>
)

export default LoaderComponent

```
